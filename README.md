# Hyperliquid Web3 Project

A simple web3 project with frontend and smart contracts.

## Structure

- `frontend/` - React frontend with Vite and ethers.js
- `contracts/` - Solidity smart contracts with Hardhat

## Getting Started

Install dependencies:
```bash
pnpm install
```

### Frontend

```bash
cd frontend
pnpm dev
```

### Contracts

Compile contracts:
```bash
cd contracts
pnpm compile
```

Deploy contracts:
```bash
cd contracts
pnpm deploy
```

Run tests:
```bash
cd contracts
pnpm test
```
